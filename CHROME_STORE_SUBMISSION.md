## 4. 권한 사유 (Privacy Practices 탭)

Chrome Web Store는 각 권한별로 사용 이유를 물어봅니다. 아래처럼 작성:

### Single Purpose
```
Convert YouTube videos into LinkedIn carousel posts using AI.
```

### activeTab
```
Read the currently active YouTube video page to detect the video title, duration, and captions for carousel generation.
```

### storage
```
Store the user's authentication token, brand settings (display name, handle, brand color, default template), and legacy screenshot captures locally.
```

### contextMenus
```
Provide a right-click menu option to open the CleanShot side panel quickly.
```

### identity
```
Allow the user to sign in with their Google account via Chrome Identity API. Only openid/email/profile scopes are used.
```

### sidePanel
```
Display the CleanShot UI as a side panel alongside YouTube videos so users can generate carousels without leaving the video.
```

### Host permissions

**youtube.com**
```
Required to extract captions from YouTube videos for AI processing. The extension reads caption text from the user's current YouTube watch page to generate carousel content.
```

**cleanshot-api.thoopring.workers.dev**
```
Our backend API that handles authentication, AI processing, subscription management, and carousel generation. All sensitive data (API keys, AI prompts) are handled server-side for security.
```

### Remote Code
```
No. The extension does not load or execute remote code. All JavaScript is bundled in the extension package.
```
(단, Google Fonts CSS를 로드하지만 이는 스타일시트이지 코드가 아니므로 해당 없음)

### Data Usage
- Sign in: Google account (email, name, profile picture)
- User data: video transcripts, generated carousel content, brand settings
- Transmitted to: cleanshot-api.thoopring.workers.dev (our backend), api.anthropic.com (AI via Cloudflare AI Gateway)
- Data selling: No, we do not sell user data.
- Data purposes: Authentication, AI processing, feature functionality

### Privacy Policy URL
```
https://thoopring.github.io/DwCanvas/privacy.html
```

## 6. 제출 절차

1. [chrome.google.com/webstore/devconsole](https://chrome.google.com/webstore/devconsole) 접속
2. 기존 CleanShot for YouTube 아이템 찾기
3. **Package** 탭 → **Upload new package** → `cleanshot-v2.0.0.zip` 업로드
4. **Store listing** 탭:
   - 제목, 요약, 상세 설명 교체
   - 스크린샷 5장 업로드
   - 카테고리: Productivity
5. **Privacy practices** 탭:
   - 위 권한 사유 복붙
   - Privacy policy URL 확인
6. **Submit for review** 클릭

검토는 보통 1~5일 소요. 정책 위반 없으면 대부분 통과.
